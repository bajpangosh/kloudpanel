- Ubuntu 22.04 LTS
- Minimum 512MB RAM
- 10GB Free Disk Space

## Quick Installation

```bash
wget -O - https://raw.githubusercontent.com/bajpangosh/kloudpanel/main/install.sh | bash
```

## Default Ports

- 8443: KloudPanel Interface
- 7080: OpenLiteSpeed Admin
- 80/443: Websites

## Components

- OpenLiteSpeed Web Server
- PHP 8.0
- MySQL 5.7
- Let's Encrypt SSL
